## 用法

若要使用嵌入向量模型，您必須提供配置來自定義其用法。如果未提供配置，將套用預設配置，並使用 `OpenAI` 作為嵌入向量模型。

有關嵌入向量模型配置的可用參數完整清單，請參閱 [配置 (Config)](./config_zh_TW)。
