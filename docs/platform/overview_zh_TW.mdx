# mem0 平台總覽

mem0 是一個記憶引擎，能保持對話的上下文，讓使用者不必重複說過的話，並讓您的代理具備回應的連貫性。mem0 平台以完全託管服務的形式提供這種體驗——無需您處理任何基礎設施工作，即可擴展、保護並豐富記憶內容。

<Tip>
  mem0 v1.0.0 推出了重新排序器 (Rerankers)、預設非同步行為以及 Azure OpenAI 支援。請在 <Link href="/changelog_zh_TW">發佈說明</Link> 中查看完整的變動清單。
</Tip>

## 為什麼這很重要

- **個人化回應**：記憶在使用者與代理之間持久化，減少了提示詞的冗餘並避免重複提問。
- **託管技術棧**：mem0 負責執行向量儲存、圖形服務和重新排序器——無需手動佈署、調優或維護。
- **企業級控制**：預設提供 SOC 2、稽核日誌和工作區治理，滿足生產環境的就緒需求。
