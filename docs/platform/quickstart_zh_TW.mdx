在 5 分鐘內開始使用 mem0 平台的託管 API。本指南將向您展示如何進行身份驗證並儲存您的第一條記憶。

## 前置需求

- mem0 平台帳號（[在此註冊](https://app.mem0.ai)）
- API 金鑰（[從控制面板獲取](https://app.mem0.ai/dashboard/settings?tab=api-keys&subtab=configuration)）
- Python 3.10+、Node.js 14+ 或 cURL

## 安裝步驟

<Steps>
<Step title="安裝 SDK">
<CodeGroup>
```bash pip
pip install mem0ai
```

```bash npm
npm install mem0ai
```

</CodeGroup>
</Step>

<Step title="設定您的 API 金鑰">
<CodeGroup>
```python Python
from mem0 import MemoryClient

client = MemoryClient(api_key="您的-API-金鑰")
````

```javascript JavaScript
import MemoryClient from 'mem0ai';
const client = new MemoryClient({ apiKey: '您的-API-金鑰' });
````

```bash cURL
export MEM0_API_KEY="您的-API-金鑰"
```

</CodeGroup>
</Step>

<Step title="新增一條記憶">
<CodeGroup>
```python Python
messages = [
    {"role": "user", "content": "我是素食者，且對堅果過敏。"},
    {"role": "assistant", "content": "沒問題！我會記住您的飲食偏好。"}
]
client.add(messages, user_id="user123")
````

```javascript JavaScript
const messages = [
    {"role": "user", "content": "我是素食者，且對堅果過敏。"},
    {"role": "assistant", "content": "沒問題！我會記住您的飲食偏好。"}
];
await client.add(messages, { user_id: "user123" });
````

```bash cURL
curl -X POST https://api.mem0.ai/v1/memories/add 
  -H "Authorization: Token $MEM0_API_KEY" 
  -H "Content-Type: application/json" 
  -d '{
    "messages": [
      {"role": "user", "content": "我是素食者，且對堅果過敏。"},
      {"role": "assistant", "content": "沒問題！我會記住您的飲食偏好。"}
    ],
    "user_id": "user123"
  }'
```

</CodeGroup>
</Step>

<Step title="搜尋記憶">
<CodeGroup>
```python Python
results = client.search("我的飲食限制有哪些？", filters={"user_id": "user123"})
print(results)
````

```javascript JavaScript
const results = await client.search("我的飲食限制有哪些？", { filters: { user_id: "user123" } });
console.log(results);
````

```bash cURL
curl -X POST https://api.mem0.ai/v1/memories/search 
  -H "Authorization: Token $MEM0_API_KEY" 
  -H "Content-Type: application/json" 
  -d '{
    "query": "我的飲食限制有哪些？",
    "filters": {"user_id": "user123"}
  }'
```

</CodeGroup>

**輸出結果：**

```json
{
  "results": [
    {
      "id": "14e1b28a-2014-40ad-ac42-69c9ef42193d",
      "memory": "對堅果過敏",
      "user_id": "user123",
      "categories": ["health"],
      "created_at": "2025-10-22T04:40:22.864647-07:00",
      "score": 0.30
    }
  ]
}
```

</Step>
</Steps>
