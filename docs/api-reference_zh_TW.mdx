## mem0 REST API

mem0 提供全面的 REST API，用於將進階記憶能力整合到您的應用程式中。透過簡單的 HTTP 請求，即可跨使用者、代理和自定義實體建立、搜尋、更新和管理記憶。

<Info>
**快速上手：** 從 [mem0 控制面板](https://app.mem0.ai/dashboard/api-keys) 獲取您的 API 金鑰，幾分鐘內即可完成您的第一次記憶操作。
</Info>

---

## 快速上手指南

只需三個簡單步驟即可開始使用 mem0 API：

1. **[新增記憶](/api-reference/memory/add-memories_zh_TW)** - 儲存來自使用者對話的資訊與上下文。
2. **[搜尋記憶](/api-reference/memory/v2-search-memories_zh_TW)** - 使用語義化搜尋檢索相關記憶。
3. **[獲取記憶](/api-reference/memory/v2-get-memories_zh_TW)** - 獲取特定實體的所有記憶。

## 身份驗證

所有 API 請求都需要使用基於 Token 的身份驗證。請在 Authorization 標頭中包含您的 API 金鑰：

```bash
Authorization: Token <您的-API-金鑰>
```

從 [mem0 控制面板](https://app.mem0.ai/dashboard/api-keys) 獲取您的 API 金鑰。

<Warning>
**請妥善保管您的 API 金鑰。** 切勿在用戶端程式碼或公開儲存庫中洩露金鑰。請僅使用環境變數並從伺服器端發送請求。
</Warning>
