# mem0 開源版總覽

mem0 開源版提供與平台相同的自適應記憶引擎，但專為需要在自有基礎設施上執行一切的團隊所打造。您將擁有技術棧、數據以及自定義功能的完整控制權。

<Tip>
  mem0 v1.0.0 推出了重新排序器 (Rerankers)、預設非同步用戶端以及 Azure OpenAI 支援。升級前請參閱 <Link href="/changelog_zh_TW">發佈說明</Link> 以獲取完整的變動概覽。
</Tip>

## mem0 開源版提供什麼

- **完整控制**：在您的環境中調優每個組件，從 LLM 到向量儲存。
- **離線就緒**：當合規性或隱私需求要求時，可將記憶保留在自有網路中。
- **可擴展的程式碼庫**：Fork 儲存庫、新增提供商並發佈自定義自動化功能。

<Info>
  請從 <Link href="/open-source/python-quickstart_zh_TW">Python 快速上手</Link>（或 Node.js 版本）開始，複製儲存庫、配置依賴項目，並在本地驗證記憶的讀取/寫入。
</Info>

## 預設組件

<Note>
  mem0 開源版開箱即用，具備合理的預設設定：
  - LLM: OpenAI `gpt-4.1-nano-2025-04-14` (透過 `OPENAI_API_KEY`)
  - 嵌入向量 (Embeddings): OpenAI `text-embedding-3-small`
  - 向量儲存: 本地 Qdrant 實例，數據儲存在 `/tmp/qdrant`
  - 歷史儲存: SQLite 資料庫，位於 `~/.mem0/history.db`
  - 重新排序器 (Reranker): 停用，直到您配置提供商。

  可透過 <Link href="/open-source/configuration_zh_TW">`Memory.from_config`</Link> 覆寫任何組件。
</Note>
