# mem0 REST API 伺服器

mem0 提供一個 REST API 伺服器（使用 FastAPI 編寫）。使用者可以透過 REST 端點執行所有操作。該 API 還包含 OpenAPI 文件，伺服器執行時可在 `/docs` 路徑存取。

## 功能

- **建立記憶**：根據使用者、代理 (Agent) 或執行 (Run) 的訊息建立記憶。
- **檢索記憶**：獲取特定使用者、代理或執行的所有記憶。
- **搜尋記憶**：根據查詢搜尋儲存的記憶。
- **更新記憶**：更新現有記憶。
- **刪除記憶**：刪除特定記憶，或刪除特定使用者、代理或執行的所有記憶。
- **重置記憶**：重置特定使用者、代理或執行的所有記憶。
- **OpenAPI 文件**：可透過 `/docs` 端點存取。

## 執行伺服器

請遵循 [說明文件](https://docs.mem0.ai/open-source/features/rest-api) 中的指令來執行伺服器。
